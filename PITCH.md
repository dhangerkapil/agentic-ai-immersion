## üÜï What's New: Azure AI Agents v2 Experience

This workshop features the **new Azure AI Agents developer experience (Agents v2)** - a major upgrade from the previous Assistants API.

### Key Improvements in Agents v2:
| Feature | Agents v2 (New) | Previous Experience |
|---------|-----------------|---------------------|
| **Architecture** | Foundry Agent Service with unified runtime | Assistants API |
| **Multi-Agent** | Built-in agent-to-agent messaging | Manual orchestration |
| **Tool Orchestration** | Server-side execution with automatic retry | Client-side handling |
| **Observability** | Full conversation tracing + App Insights | Limited visibility |
| **Trust & Safety** | Integrated content filters, XPIA protection | Basic filters |
| **Enterprise** | BYOS, VNet, RBAC, Cosmos DB for state | Limited options |

### üìñ Upgrade Documentation:
- [Upgrading to the new agents developer experience](https://learn.microsoft.com/en-us/azure/ai-foundry/agents/concepts/agents-upgrade)
- [What is Foundry Agent Service?](https://learn.microsoft.com/en-us/azure/ai-foundry/agents/overview)
- [Environment Setup for Agents v2](https://learn.microsoft.com/en-us/azure/ai-foundry/agents/environment-setup)
